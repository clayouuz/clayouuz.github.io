# 1：项目目标

1：实现学习资料的代际传承和班级之间共享

2：实现基层学科论坛（试验）

3：便于教师与学生之间资料传递（例如教师 ppt 或讲义）

4：方便住宿生最新资料获取

5：评教系统（试验）

# 资料池

包含资料的上传，下载，评论，举报功能。（不能重复（网站警告），不能违反社会主义核心价值观（处罚））

资料按上传类型分为：

毕业校友上传资料（免费）；

学生自行上传资料（本班级不限次数，其他班级下载消耗积分）；

教师上传资料（仅代课班级可见，不限次数）；

公众号网站爬取资料（积分）。

对于暂时不存在的资料可以选择订阅，有相关资源后推送通知

资料按种类分为：

试卷及其答案；压轴题（或竞赛）及其解答；电子教参；知识点题型汇总；软件工具，讲解视频链接（试验）。

资料推送方式：

试卷（优先时间顺序，优先带答案）；折叠没有答案的试卷，手动展开；

其余优先下载量。

按考区分开

已下载用户可进行点评，包括试题解析，难易程度，题面质量

l 试验：增加知识点交流板块

1：班级可以上传图片版问题并开新帖子，添加学科 tag，帖子可以标记为已解决 tag，可以在上传一日之内编辑帖子（包括添加简介，添加图片，添加题解等）。tag 包含（学科，难度，年级（auto））

2：当帖子三天之内没有新的浏览者，这个帖子将不再出现在推荐列表上，但仍然可以被通过关键字搜索到。

暂定每周集体使用一次，时间机制修改为一周。

3：班级可以自行选择屏蔽某些已经浏览过的帖子。被屏蔽的帖子不会出现在班级的推荐列表中，但仍然可以被通过关键字搜索到。

4：帖子构成：

必须部分：图片版题面，提问班级，上传时间，是否解决，学科 tag（题目和难度），回答人（精确到班），回答时间，回答内容（必须图片），收藏按钮。

可选部分：是否有老师指导，题目来源。

5：帖子与积分挂钩，将在下一板块详细阐述。

6：发帖人和管理员可以自行选择隐藏帖子，帖子在隐藏之后不能被浏览和举报，同样的，通过此帖子获得的积分也将被扣除，同时自行隐藏帖子将扣除发帖人部分积分。

7：帖子排序方式：

1：已解决的帖子：根据回答数量，浏览量，赞同量和是否有教师参与进行优先级排序。

2：未解决的帖子：根据发表时间按最新的靠前排序。

3. 回答内容：题解（必须），题目点评（可选），知识点易错点（可选），感想（可选）

8：班级提问问题同时自动以邮件格式发送给本班相应科任老师。

9：每日一题系统（试验）

10：管理条例 (拟)：

1：不允许班级在工作时间发帖（包括提问和回答）

暂定允许的时间段：周一到周六值日时间和中午时间段：不限制数量。

周日全天：限制每小时不能发布超过 3 次。

2：帖子在被举报之后将会由管理员审查，审查通过后如果情况属实，扣大分，取消本班在从发帖日起 3 天内发帖权限与获得积分权限，但是仍能浏览，收藏，下载资料。将情况通报给班主任处理。

3：每班每帖只能使用一次赞同，不能赞同自己的帖子（技术实现）。

# 积分管理系统

1：什么是积分？

积分是一个班级的账户，积分可以用于下载资料和提出问题，积分将在每学年末重置。

2：积分获得的条件

    1. 在班级账户不处于处罚状态时，每天每班将会自动获得一定积分。
    2. 班级在上传共享资料时可以为资料标价，每有一个下载者，其支付的积分将自动进入本班账户。
    3. 被点赞可以获得额外积分，被点踩就是摆设，充分调动学生的学习积极性。
    4. 班级在回答问题时，每获得一个赞同将获得一定积分。
    5. 其余可能的奖励方式（每学期评比奖励，月考奖励）。

3：积分扣除的条件

    1. 班级可以消耗积分使问题保持在推荐列表上。
    2. 班级消耗积分下载有偿共享文件。
    3. 举报失败扣除积分 *，* 醒目的警告。
    4. 其他所有可能的扣分方式，解释权归管理人员所有。

# 教师资料上传体系

  1. 教师可以用邮箱注册账号，教师认证通过之后获得对所有资料的下载权限和所有讨论的编辑隐藏权限。
  2. 教师认证：由学校负责
  3. 教师在注册时可以填写自己的教学班级（选）。
  4. 填写教学班级的教师可以执行以下操作：

向服务器信箱发送邮件，服务器将自动接受此邮件并且上传到本班级的资料区，并将邮件内容作为简介。在教师设置的时间内（默认不限时），教师负责的教学班级可以下载这个文件并且对其他所有班级不可见。对于没有附件的邮件，正文信息将以公告形式推送给教学班。

  1. 教师可以选择令所负责教学班的所有行为生成日报以邮件形式发送到教师信箱。

发送后可选择是否批量推送到相应班级邮箱
